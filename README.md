# 📘 Facebook Login Page Clone

A front-end clone of the Facebook login and signup page built using HTML, CSS, and JavaScript (jQuery). This project focuses on recreating the UI and implementing form validation.

## 🚀 Features

* Facebook-style login header
* Signup form with validation
* Email or phone number validation
* Password validation (length constraints)
* Date of birth selection
* Gender selection
* Clean and structured layout
* Basic responsive design using Flexbox

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (jQuery)
* jQuery Validation Plugin

## ✅ Form Validation

The signup form includes validation for:

* First Name (minimum 3 characters)
* Last Name (minimum 2 characters)
* Email or Phone Number (custom validation)
* Password (6–12 characters)
* Date of Birth (all fields required)
* Gender (required selection)
